# Tokens

## Color

- `color-background`: app background, `#0B1720`.
- `color-surface`: base panels and inputs.
- `color-surface-elevated`: cards, modals, raised panels.
- `color-surface-muted`: metric cards and darker panels.
- `color-surface-subtle`: hover fills and subtle containers.
- `color-border`: default border.
- `color-border-strong`: emphasized border.
- `color-primary`: primary action.
- `color-primary-hover`: primary hover.
- `color-primary-subtle`: active navigation and secondary action fills.
- `color-text`: primary text.
- `color-text-muted`: secondary text.
- `color-text-subtle`: low-emphasis text.
- `color-text-disabled`: disabled text.
- `color-success`, `color-warning`, `color-danger`, `color-info`: semantic feedback.

## Typography

All typography uses `Rethink Sans`.

- `display-lg`: 48/56 Bold.
- `display-md`: 40/48 Bold.
- `heading-lg`: 32/40 Bold.
- `heading-md`: 24/32 SemiBold.
- `title-lg`: 20/30 SemiBold.
- `title-md`: 18/28 SemiBold.
- `body-lg`: 16/24 Regular.
- `body-md`: 14/20 Regular.
- `body-sm`: 12/16 Regular.
- `label-md`: 14/20 Medium.
- `label-sm`: 12/16 Medium.
- `caption`: 10/14 Regular.

## Spacing

Use the 4px scale: `0`, `4`, `8`, `12`, `16`, `20`, `24`, `32`, `40`, `48`.

## Radius

- `xs`: 4px.
- `sm`: 6px.
- `md`: 8px.
- `lg`: 10px.
- `xl`: 12px.
- `2xl`: 14px.
- `panel`: 20px.
- `full`: 9999px.

## Motion

- `fast`: 120ms.
- `base`: 180ms.
- `slow`: 240ms.

## Shadows

- `shadow/elevation/sm`: compact panels, dropdowns, lightweight toasts.
- `shadow/elevation/md`: modals, raised panels, promoted cards.
- `shadow/focus/primary`: keyboard focus and high-emphasis active states.

## Opacity

- `opacity/disabled`: disabled actions and unavailable data.
- `opacity/overlay`: modal and blocking overlay layers.

## Z-Index

- `z/base`: default content plane.
- `z/sticky`: sticky table headers, floating utility elements.
- `z/modal`: dialogs, overlays, blocking surfaces.

## Layout

- `layout/screen/desktop`: canonical desktop screen width.
- `layout/sidebar/width`: production sidebar width.
- `layout/content/max`: standard max content region.
- `layout/card/iconBox/lg`: icon container for large KPI and shortcut cards.
- `layout/card/iconBox/md`: icon container for compact KPI cards.

## Typography Support Tokens

- `font/size/display-lg`
- `font/size/heading-lg`
- `font/size/body-md`
- `font/line/body-md`

These support the Figma variable layer and should stay aligned with the official text styles.
